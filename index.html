  • 3D Positioning: Dynamic depth effects

Customization Options

The carousel can be customized through data attributes:

<div class="techno-orb-carousel" 
     data-radius="400" 
     data-height="300" 
     data-max-visible="5" 
     data-auto-rotate="true" 
     data-auto-rotate-delay="4000"
     data-show-controls="true"
     data-show-dots="true">

Parameter Reference

  • data-radius: Radius of the carousel orbit (px)
  • data-height: Height positioning (px)
  • data-max-visible: Number of visible items
  • data-auto-rotate: Enable/disable auto-rotation (true/false)
  • data-auto-rotate-delay: Rotation delay in milliseconds
  • data-show-controls: Show navigation buttons (true/false)
  • data-show-dots: Show dot indicators (true/false)

Adding Portfolio Projects

To add projects to the carousel, edit the page-portfolio.php template and add your project content within the carousel structure.

Common Issues

Solution:

  1. Ensure JavaScript is enabled in your browser
  2. Check browser console for JavaScript errors
  3. Verify carousel-sphere.js file is properly enqueued
  4. Clear browser cache and WordPress cache

‚ö†Ô∏è Styles Not Loading

Solution:

  2. Check file permissions on theme files
  3. Verify CSS files are properly linked
  4. Deactivate other plugins temporarily

‚ö†Ô∏è Customizer Changes Not Saving

Solution:

  2. Disable other themes temporarily
  3. Increase PHP max_execution_time
  4. Contact hosting provider if issues persist

  • Plugins: Minimize number of active plugins
  • Hosting: Ensure adequate server resources
