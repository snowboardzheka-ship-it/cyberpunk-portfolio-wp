== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Appearance > Themes
3. Click "Add New" then "Upload Theme"
4. Select the theme.zip file and click "Install Now"
5. After installation, click "Activate" to enable the theme

= Manual Installation =

1. Download the theme.zip file
2. Extract the contents to your computer
3. Upload the 'my-portfolio' folder to /wp-content/themes/
4. Log in to your WordPress admin panel
5. Navigate to Appearance > Themes and activate "My Portfolio"

= Requirements =

* WordPress 5.0 or higher
* PHP 7.4 or higher
* Modern web browser with JavaScript enabled
* Minimum 512MB RAM for optimal performance

== Theme Setup ==

After activation, follow these steps:

1. **Customize Your Site**
   - Go to Appearance > Customize
   - Set your site title, tagline, and upload a logo
   - Configure colors in "Color Scheme" section
   - Add your social media links

2. **Create Portfolio Pages**
   - Create a new page called "Portfolio"
   - Set page template to "Portfolio Template"
   - Publish the page
   - Add the page to your main menu

3. **Set Up Navigation**
   - Go to Appearance > Menus
   - Create a new menu called "Main Menu"
   - Add your pages and links
   - Assign menu to "Primary Menu" location

4. **Configure Homepage**
   - Go to Settings > Reading
   - Set "Front page displays" to "A static page"
   - Create and select a "Home" page

== Frequently Asked Questions ==

= Do I need coding skills to use this theme? =

No coding skills required! The theme includes comprehensive WordPress Customizer options for all major customizations. You can change colors, text, and add content through the WordPress admin interface.

= Is the 3D carousel mobile-friendly? =

Yes! The Techno Orb Carousel is fully responsive and includes touch/swipe support for mobile devices. The 3D effects automatically adapt to screen size for optimal performance.

= Can I change the color scheme? =

Absolutely! Use the WordPress Customizer (Appearance > Customize) to change primary, secondary, and accent colors in real-time with live preview.

= Does it work with page builders? =

The theme is built with standard WordPress structure and works well with most page builders. However, it's optimized for its built-in templates and custom layouts.

= Is it SEO optimized? =

Yes! The theme includes:
- Semantic HTML5 markup
- Optimized meta tags
- Fast loading times
- Clean code structure
- Social media integration

= Can I import demo content? =

The theme includes built-in demo content that automatically appears when you create pages with the provided templates. No separate import process required.

= Does it support multiple languages? =

The theme is translation-ready and includes a .pot file for easy translation. It also supports RTL languages.

= What browsers are supported? =

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Internet Explorer 11+ (with limited effects)

= How do I add portfolio projects? =

Create new pages using the "Portfolio Template" or add projects directly to the carousel by modifying the page-portfolio.php template with your content.

= Can I customize the animations? =

Yes! The theme uses CSS custom properties (variables) that make it easy to customize animation timing, colors, and effects. Advanced users can modify the JavaScript files for deeper customization.

= Is the theme GDPR compliant? =

The theme itself doesn't collect any personal data. However, when using contact forms or analytics, make sure to comply with your local privacy regulations.
